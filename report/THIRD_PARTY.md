# Third-Party Report Assets

This report vendors a small set of assets so the Practical Work PDF can be
rebuilt without depending on an external LaTeX template installation.

| Asset | Source | License |
|---|---|---|
| `jkureport.sty`, `logos/*.pdf` | `michaelroland/jku-templates-report-latex` release `v2.2` | Mozilla Public License 2.0; see `TEMPLATE_LICENSE` and the header in `jkureport.sty` |
| `fonts/PublicSans-*.ttf` | Public Sans | SIL Open Font License 1.1; see `fonts/PublicSans-LICENSE.txt` |
| `fonts/Merriweather-*.ttf` | Merriweather | SIL Open Font License 1.1; see `fonts/Merriweather-LICENSE.txt` |
| `fonts/Inconsolata*.ttf` | Inconsolata | SIL Open Font License 1.1; see `fonts/Inconsolata-LICENSE.txt` and `fonts/InconsolataCondensed-LICENSE.txt` |
| `ACM-Reference-Format.bst` | ACM bibliography style by Nelson H. F. Beebe, Boris Veytsman, and Gerald Murray | Public domain, as stated in the file header |
| `ACM-Reference-Format.bbx`, `.cbx`, `.dbx` | BibLaTeX compatibility files used with the ACM reference format | Vendored with the report template assets; kept only to make the bibliography build reproducible |
